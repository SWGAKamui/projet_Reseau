	    PONG WARS
Que la force du pong soit avec vous !

Tout d'abord se placer dans les sources :
cd src

Pour compiler le projet :
      make

Pour lancer les tests :
      make test

Pour commencer à jouer (un joueur) :
      make run
      
Pour lancer le pong en réseau pour le player 1 :
      make runP1
      
Pour lancer le pong en réseau pour le player 2 :
      make runP2
      
Pour lancer le pong en réseau pour le player 3 :
      make runP3

Pour lancer le pong en réseau pour le player 4 :
      make runP4

Pour lancer le pong en réseau de manière générale :
     java pong.Main localPort host port
