
Pour compiler le projet :
      make

Pour lancer le pong :
      make run
      
Pour lancer le pong en réseau:
      make run P1
      
Pour lancer le pong en réseau pour le premier joueur:
      make run P2


Pour lancer les tests :
      make test
      
      


Aborescence du projet :
      - image :
            - ball.png
            - racket.png
            
      - pong :
            - gui :
                    - Ball.java
                  - Player.java
                  - PlayerID.java
                  - Pong.java
                  - PongItem.java
                  - Racket.java
                  - Window.java
            - util :
                    - RandomNumber.java
            - Main.java
            
      - test :
            - Test.java
            
      - readme.txt
